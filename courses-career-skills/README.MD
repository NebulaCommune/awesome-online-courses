## 📘 本仓库内容
本仓库收录 **直接面向就业** 的课程：
- 前端就业实战：Vue、React、工程化、面试
- 后端就业实战：Java、Spring Boot、微服务、数据库
- 嵌入式 / 硬件就业：STM32、嵌入式Linux、项目开发
- AI / 数据分析就业：项目实战、业务落地
- 通用就业技能：简历、面试、职场沟通

所有课程强调：**能上手、能写进简历、能应对面试**。
