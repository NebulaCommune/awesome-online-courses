## 📌 课程提交格式

推荐课程时，请统一使用以下 Markdown 格式，方便维护和阅读：

```markdown
### 课程名称
- 平台：Bilibili / Coursera / 网易云课堂 / 其他
- 链接：[点击查看](https://example.com)
- 适合人群：零基础 / 进阶 / 就业向
- 推荐理由：简短说明课程亮点、学习收获
- 标签：#标签1 #标签2 #标签3
```

**示例：**

```markdown
### C语言程序设计（浙江大学）
- 平台：中国大学MOOC
- 链接：[https://www.icourse163.org/course/ZJU-9001](https://www.icourse163.org/course/ZJU-9001)
- 适合人群：零基础入门
- 推荐理由：体系完整、讲解清晰，配套实验丰富，是公认的 C 语言入门经典。
- 标签：#C语言 #计算机基础 #入门
```

## 🤝 欢迎贡献

你可以通过以下方式参与本仓库：

1.  **Star 本仓库**，让更多人发现优质资源。
2.  **Fork 仓库**，在对应分类中添加你认可的课程。
3.  提交 **Pull Request**，等待维护者审核合并。
4.  在 **Issues** 中推荐课程、反馈失效链接或提出改进建议。

> 注意：提交前请确保课程信息准确、链接可访问，且符合仓库收录原则（体系完整、口碑良好）。
