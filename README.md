# awesome-online-courses

精选优质在线课程，助力高效学习与技能成长。

这里是 **NebulaCommune 星隅共学社** 维护的公开课程清单体系，按「学习目标」分类整理，只收录真正学过、口碑过硬、体系完整的优质网课，帮你精准找到匹配需求的课程，少走弯路。

---

## 📘 项目说明

本项目按「学习目标/场景」拆分仓库，让你能快速定位符合自身需求的课程：
- 基础课程：打牢知识根基，适合零基础/想补基础的学习者
- 就业技能：聚焦职场实战，适合想找工作/职场进阶的开发者
- 考证备考：专攻各类证书，适合有考证需求的人群
- 所有内容由社区成员共同维护，坚持「宁缺毋滥，只推荐有价值的资源」

## 📌 贡献指南

### 1. 贡献方式
- **Star 相关仓库**：让更多人发现优质资源
- **Fork 对应仓库**：补充你认可的课程
- **提交 Pull Request**：等待维护者审核合并
- **Issues 反馈**：推荐课程/反馈失效链接/提改进建议

### 2. 统一提交格式
所有仓库的课程提交均遵循以下格式，确保可读性：
```markdown
### 课程名称
- 平台：Bilibili / Coursera / 网易云课堂 / 其他
- 链接：[点击查看](https://example.com)
- 适合人群：零基础 / 进阶 / 备考人群 / 求职人群
- 推荐理由：简短说明课程亮点、学习收获
- 标签：#标签1 #标签2 #标签3
